# Calculator & Calendar Web App

A simple and clean web application that combines a Calculator and a Calendar in one interface. This project is beginner-friendly and demonstrates basic HTML, CSS, and JavaScript concepts with a structured and professional layout.

## Description

This project includes two main components in one application:
- Calculator for performing basic arithmetic operations
- Calendar for viewing dates and navigating months

Both features are designed to be easy to use and visually organized.

## Features

- Perform basic calculations (addition, subtraction, multiplication, division)
- Instant result display with clear functionality
- View current month and year in calendar
- Highlight current date
- Navigate between months
- Clean and simple user interface

## Technologies Used

- HTML
- CSS
- JavaScript
